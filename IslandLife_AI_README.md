# 🌴 IslandLife AI

> **An AI-Powered Cozy Life Collection & Inventory Management Platform**

## 📖 Overview

IslandLife AI is a full-stack AI-powered web application inspired by
cozy life-simulation games. It combines modern UI/UX with Artificial
Intelligence, Computer Vision, Semantic Search, and Machine Learning.

## ✨ Features

-   AI-powered item recommendations
-   Image recognition with TensorFlow/OpenCV
-   Semantic search using Sentence Transformers + FAISS
-   Smart inventory & reserve storage
-   Puzzle system with rewards
-   Responsive mobile-first UI
-   Dark/Light mode
-   Progressive Web App (PWA)

## 🚀 Installation

### Clone

``` bash
git clone https://github.com/yourusername/IslandLife-AI.git
cd IslandLife-AI
```

### Frontend

``` bash
cd frontend
npm install
npm run dev
```

### Backend

``` bash
cd backend
pip install -r requirements.txt
uvicorn app.main:app --reload
```
